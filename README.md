# welcome to monty!

> code open source Python packages, publish them to PyPI, and earn Venom Coins V to get stuff! That's the entire gig. You ship code, we ship loot!


This project is written with Typescript/TailwindCSS for styling, with the help of the NextJS framework :). The styling is inspired by cartoony designs :).
It only has the main page ('/') and FAQ ('/faq').

To run this app, git clone, install libraries and npm run it!

```
git clone https://github.com/xndadelin/monty
cd monty
npm install
npm run dev
```

There you go!

If you want to contribute, fork it, write clear commit messages and explain the changes and make a PR!
